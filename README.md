# Free Cup Holder for Linux
Very useful utility.

## Required packages
- GCC
- Make

## Build Instructions
### Debian:
1. Run `sudo apt-get install make gcc`
2. Run `make all`
This should work for any other distro, and even other *nix systems (minus apt)

## Installation
### Debian:
1. Build it
2. Run `sudo make install`

## Uninstallation
1. Run `sudo make uninstall`
